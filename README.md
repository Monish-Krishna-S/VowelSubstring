# VowelSubstring
The problem is to find longest substring between two vowels in a given string of lowercase letters. 
    If there is more than one longest substrings, you have to print them too.
    
    Sample I/Os:
    
    Input 1:
        single
    Output 1:
        ngl
    
    Input 2:
        america
    Output 2:
        m,r,c
    
    Input 3:
        alphabet
    Output 3:
        lph
